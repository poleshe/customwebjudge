Crear un nuevo test


    Crear un nuevo test está formado por tres pasos. Una vez terminados, el nuevo test aparecerá en la lista de tests y podrá ser modificado. Pasos:

    · El primer paso es crear los datos básicos. Debes poner un nombre al test, quien lo ha creado, y una explicación de lo que se ha de conseguir. Es aconsejable añadir casos de prueba con datos de entrada / salida.

    · Después, tendrás que subir el archivo base HTML que contiene la plantilla para ejecutar el test. Mas info en la siguiente página.

    · Por ultimo, se añaden los pasos que deberá seguir el test. Ten en cuenta de que se debe completar el test correctamente para poder ser subido a la web, así que prepara tu archivo con la respuesta.
